# EnXi Marketing UI Kit — The Website

Recreates the EnXi marketing site: homepage hero (The Stated Fact), a case-study post-mortem layout, and the "Not good for" page. Uses the Ledger Grid (14 columns, marginalia column) and only lets GT Sectra appear once per page.

## Components
- `SiteHeader.jsx` · `SiteFooter.jsx`
- `Hero.jsx` — the customer quote on bone substrate, dateline, hairline rule
- `Lede.jsx` · `Broadsheet.jsx` — 5-beat editorial layout with marginalia
- `Marginalia.jsx` — right-rail notes in Söhne Mono, `ink/40`
- `PullQuote.jsx` · `HairlineRule.jsx` · `SourceCite.jsx`
- `NotForPage.jsx` — the trust artifact

## Notes
- Sectra substitute: DM Serif Display. Söhne substitute: Inter. Tiempos substitute: Source Serif 4.
